# Finanzas-Corporativas-Plan-de-ferreteria
Análisis de viabilidad financiera para apertura de sucursal de ferretería industrial. Incluye estudio de mercado, proyección de ventas y costos, modelado de flujo de fondos a 5 años y evaluación con VAN, TIR y payback. Se incorpora análisis de sensibilidad y estructura de financiamiento para respaldar decisiones estratégicas basadas en datos.
